Support / 支援

Support

Get help with installation, theme generation, preview assets, fonts, and manual import for Codex Material Themes.

Current release / 目前版本：1.0.2

Contact / 聯絡方式

Email: robin@bud4.net

Issues: github.com/robinli/codex-material-themes/issues

When reporting a problem, include your operating system, Codex surface, plugin version, prompt, expected result, and actual result. 請附上作業系統、Codex 使用介面、Plugin 版本、提示詞、預期及實際結果。

Installation / 安裝

  1. Install Codex Material Themes from your Plugins Directory.
  2. Start a new conversation after installation or update.
  3. Ask for a theme recommendation or invoke the bundled theme skill.

安裝或更新後請開啟新對話，讓 Codex 載入最新 Skill。

Theme import / 主題匯入

  1. Keep the complete codex-theme-v1: value on one line.
  2. Open Settings → Appearance → Import.
  3. Paste the value and select Import theme.

Plugin 不會自行修改 Codex 設定；匯入動作由使用者完成。

Preview images / 預覽圖片

Version 1.0.2 displays previews from public HTTPS image URLs. If a preview cannot be embedded, open the clickable HTTPS preview link in the response, review the image, and continue the same confirmation flow.

1.0.2 會透過公開 HTTPS 圖片網址顯示預覽。若介面無法內嵌圖片，請直接開啟回覆中的可點擊 HTTPS 預覽連結，查看圖片後繼續原本的確認流程。

After an update / 更新後

If an older workflow still appears, remove and reinstall the plugin, fully quit Codex, reopen it, and test in a new conversation.

若仍出現舊流程，請移除並重新安裝 Plugin、完整結束 Codex 後重開，再於新對話測試。

Fonts / 字型

Font names are passed through to Codex. Make sure the selected UI and code fonts are installed on your computer; otherwise choose a portable fallback such as sans-serif or monospace.

請確認選擇的 UI 與程式碼字型已安裝；若無，可使用通用替代字型。

Scope / 支援範圍

Support covers the bundled skill, theme catalogue, generated setting format, preview assets, and this website. Account, billing, and core ChatGPT or Codex platform issues should be directed to the platform provider.

支援範圍包括內附 Skill、主題目錄、設定格式、預覽資產及本網站；帳號、計費與平台本身問題請洽平台提供者。
